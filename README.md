## Run Locally

Clone the project

```bash
  git clone https://github.com/d-a-ve/portfolio.git
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
